## 1.ssh连接
刚买的服务器需要重置root密码。在实例的控制台上在线远程连接，然后点击重置密码。该密码即为root密码。随后即可通过ssh user@ipaddr连接服务器。

### 设置密钥连接
首先在本地找到密钥对：
```bash
cat ~/.ssh/id_rsa.pub
```
复制下来，随后
```bash
## 连接到服务器：
ssh user@ipaddr
## 在远程操作：
mkdir -p ~/.ssh
chmod 700 ~/.ssh
touch ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
vi ~/.ssh/authorized_keys
```
把cat出来的公钥复制到服务器上，即可完成密钥连接设置。
